## Installation

First, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2. Install dependencies:
    ```bash
    npm install
    ```
